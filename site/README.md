# Site institucional — Bee Operações Online

## Descrição
Página única inspirada no estilo do Entire.io. Cria um layout elegante em blocos com hero, cards de valor, provas e CTA final, destacando o posicionamento de diretora de agentes inline.

## Estrutura
- `index.html`: HTML com hero, seções de valor, provas e CTA final.
- `styles.css`: Estilos com gradientes, tipografia Space Grotesk e interações suaves.

## Como visualizar
1. Abra o arquivo `index.html` em um navegador (duplo clique ou `open`/`xdg-open`).
2. Pode servir via GitHub Pages se subir em branch `gh-pages` ou usando qualquer servidor de arquivos estáticos.
3. Para testes locais com Node: `npx serve programmer-agent/site/` (opcional).
